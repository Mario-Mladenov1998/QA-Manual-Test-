
# Login Test Cases

## ✅ Positive Test
**Title**: Valid Login  
**Steps**:  
1. Open login page  
2. Enter valid username and password  
3. Click login  
**Expected Result**: User is redirected to dashboard

## ❌ Negative Test
**Title**: Invalid Password  
**Steps**:  
1. Open login page  
2. Enter valid username and wrong password  
3. Click login  
**Expected Result**: Error message is shown
